Página web dedicada a la visualización de datos meteorológicos en tiempo real centrada en la península ibérica:

index.html: Comparación temporal de modelos de predicción AROME HD, ICON-EU, GFS, con una pequeña sección dedicada a ECMWF.
Observaciones.html: Observaciones en tiempo real de temperatura, radar, satélite, viento, etc.
aeronautica.html: Sección dedicada a la presentación de datos meteorológicos aeronáuticos.
